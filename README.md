# Emergency Help Portal

Yeh project har tarah ki emergency ke liye important helpline numbers aur information provide karta hai.

## Included Sections
- Disaster Management Helpline
- Women Helpline
- Child Helpline
- Medical Emergency
- Fire & Police 
- Blood Bank Information

## Purpose
School project ke liye ek working web page ya information portal jisme students aur users emergency me help numbers easily dekh saken.
